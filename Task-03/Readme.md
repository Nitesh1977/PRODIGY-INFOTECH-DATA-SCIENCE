# Task 3 - Decision Tree Classification on Bank Marketing Dataset

## About This Project
In this task, I worked on the Bank Marketing dataset and built a Decision Tree Classification model to predict whether a customer will purchase a product or service.

## What I Did
- Loaded and explored the dataset
- Checked and handled categorical data
- Converted text data into numerical format using Label Encoding
- Split the dataset into training and testing data
- Trained a Decision Tree Classifier model
- Made predictions using the model
- Evaluated the model accuracy

## Tools & Libraries Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Concepts Used
- Data preprocessing
- Label Encoding
- Train-Test Split
- Decision Tree Classification
- Model Prediction
- Accuracy Evaluation

## What I Learned
Through this task, I learned:

A Decision Tree model was created to predict customer responses based on the given data.
