# Task 2 - Titanic Dataset Exploratory Data Analysis

## About This Project
In this task, I worked on the Titanic dataset and performed Exploratory Data Analysis (EDA) to understand the data better and discover useful patterns.

## What I Did
- Loaded and explored the dataset
- Checked missing values
- Cleaned the data
- Analyzed passenger survival based on gender and passenger class
- Created different visualizations using graphs

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations Created
- Survival count by gender
- Survival count by passenger class
- Age distribution graph
- Correlation heatmap

## What I Learned
Through this task, I learned:
- Data cleaning techniques
- Handling missing values
- Exploratory Data Analysis (EDA)
- Data visualization using Seaborn and Matplotlib
- Understanding relationships between different features
