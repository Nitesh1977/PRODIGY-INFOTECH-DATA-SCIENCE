# Task 1 - Population Data Visualization

## About This Project
In this task, I worked on a world population dataset and created a visualization of the top 10 most populated countries in 2022.

## What I Did

- Explored the data
- Sorted countries based on population
- Selected the top 10 countries
- Created a bar chart for visualization

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib

## What I Learned
Through this task, I learned:
- How to work with CSV datasets
- Data filtering and sorting
- Creating graphs using Matplotlib
- Basic data visualization techniques

## Output
A bar chart showing the top 10 most populated countries.
