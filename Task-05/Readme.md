# Task 5 - US Accidents Data Analysis

## About This Project
In this task, I worked on the US Accidents dataset and performed data analysis to identify patterns related to road accidents.

## What I Did
- Loaded and explored the dataset
- Checked and handled missing values
- Selected important columns for analysis
- Analyzed accident trends based on time, weather conditions, and states
- Created different visualizations using graphs

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Accidents by hour
- Weather condition analysis
- Top states with accidents
- Accident severity analysis

## What I Learned
Through this task, I learned:
- Data cleaning techniques
- Working with large datasets
- Handling missing values
- Time-based data analysis
- Data visualization using graphs
- Finding patterns and insights from real-world data

## Output
Different graphs and insights were generated to better understand accident patterns in the US.
